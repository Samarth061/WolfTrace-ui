# Download Dune Rise Font

To complete the font setup, you need to download the Dune Rise font file:

## Step 1: Download
Visit one of these links:
- **DaFont**: https://www.dafont.com/dune-rise.font
- **1001 Fonts**: https://www.1001fonts.com/dune-rise-font.html

Click the "Download" button.

## Step 2: Extract
The download will be a `.zip` file. Extract it to find the font file.

## Step 3: Copy to Project
Copy the `DuneRise.ttf` file to this directory:
```
/public/fonts/DuneRise.ttf
```

## Step 4: Restart
Restart your Next.js dev server:
```bash
# Stop the server (Ctrl+C), then:
npm run dev
```

## Done!
Once the font file is in place, Dune Rise will be applied everywhere on your website.
